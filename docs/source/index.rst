Mapping Environmentally Sensitive Assets (MESA) tutorial
==========================================================

A guide to using the MESA tool developed by UNEP-WCMC and NEA. 

Please note this document is based on the previously published ‘MESA: Mapping Environmentally Sensitive Assets’ (NEA and UNEP-WCMC, 2020), which has since been divided into two parts: a Guidance Document and this Tutorial. The Guidance Document is titled: UNEP-WCMC and NEA (2025) MESA: Mapping Environmentally Sensitive Assets – A spatial approach to support environmental management. Version 2.
